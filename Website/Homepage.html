ΠΡΩΘΥΠΟΥΡΓΟΣ

Επισκόπηση εθνικών δεικτών και στρατηγικών σεναρίων.

  1. Log in with secure government credentials.
  2. Select national priority (economy, education, health, etc.).
  3. Review high level dashboards and risk alerts.
  4. Export summary for cabinet meeting.

  2. Analyze performance and funding indicators.
  3. Run “what if” policy scenarios.
  4. Generate report for implementation teams.

  3. View projections and sustainability indicators.
  4. Download detailed budget brief.
